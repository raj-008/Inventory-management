## About The Project

An **Inventory Management System** allows you to:

- Store details about various products
- Generate invoices
- Manage stock efficiently

This project is useful for small businesses or personal use to track inventory and sales.

## Built With

- [React.js](https://reactjs.org/)
- [Node.js](https://nodejs.org/)
- [Express.js](https://expressjs.com/)
- [MongoDB](https://www.mongodb.com/)

## Installation

Follow these steps to run the project locally:

1. **Clone the repository**  
   ```sh
   git clone https://github.com/raj-008/Inventory-management.git

 2. **Install dependencies**
    <br>
    Navigate into both client and server folders and run: 
     ```sh
     npm install

 4. **Clone the repository**  
    Add your environment variables (e.g., MongoDB URI, PORT, JWT secret).

 5. **Start the Backend**  
     ```sh
     npm start

 6. **Start the Frontend**  
     ```sh
     npm run build
